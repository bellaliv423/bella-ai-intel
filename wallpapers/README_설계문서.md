## v2 — Bold Invader (v2_bold_invader.html)

**컨셉**: Claude 선글라스 캐릭터 스타일, 크고 뚜렷하게!

### 개선 포인트 (드림팀 회의 결과)
1. **크기 2배+**: 12x12 → 16x16 픽셀, 표시 150px+
2. **고유 실루엣**: 각자 귀/몸통/팔 다르게
3. **고유 아이템**: 캐릭터당 1~2개 식별 아이템
4. **이름표 크게**: 10px+ 밝은 색상
5. **배경 개선**: 약간 밝게 (#1a1a19), 팀별 섹션 구분

## v3 설계 — Space Invader (다음 버전!)

**컨셉**: Claude 공식 캐릭터처럼 각진 스페이스 인베이더 스타일!

### v2 → v3 핵심 개선 4가지

1. **실루엣 차별화**
   - 햄스터: 둥근 귀 + 네모 몸통 + 짧은 다리 2개
   - 퍼피: 늘어진 귀(삼각형) + 네모 몸통 + 짧은 다리 4개
   - **지금 문제**: 둘 다 동글동글해서 구분 안 됨
   - **목표**: 실루엣만 봐도 "이건 햄스터, 이건 퍼피!" 알 수 있게

2. **아이템 2~3배 크게**
   - v2: 아이템 1~2px → 안 보임
   - v3: 아이템 3~4px + 몸 옆에 크게 배치
   - 골디 왕관: 머리 위 3줄로 뚜렷하게
   - 윈디 안경: 얼굴 절반 차지하게
   - 도비 선글라스: Claude 공식 캐릭터처럼 넓게!

3. **어두운 캐릭터 아웃라인 밝게**
   - 헌터(#424242), 도비(#37474F), 테디(#795548) → 배경에 묻힘
   - **해결**: 밝은 아웃라인(#888) 또는 글로우 이펙트 추가
   - 또는 캐릭터 아래 밝은 그림자/발판

4. **Claude 스페이스 인베이더 스타일 참조**
   - 원본: 각진 네모 머리 + 네모 몸통 + 짧은 다리 + 큰 눈
   - 선글라스 버전처럼 눈 영역을 크게 (얼굴의 40%)
   - 입: `^ㅅ^` 또는 `ㅅ` 네모 미소
   - **다리: 2~3칸 짧은 네모 다리** (지금은 1칸이라 약함)

### v3 캐릭터 실루엣 가이드 (ASCII 참고)

```
햄스터 기본형:          퍼피 기본형:
   ▓▓                  ▓▓      ▓▓    (늘어진 귀!)
  ▓▓▓▓                 ▓▓▓▓▓▓▓▓▓▓
  ▓●●▓  (큰 눈)        ▓▓●●▓▓●●▓▓
  ▓▓▓▓                 ▓▓▓▓▓▓▓▓▓▓
  ▓▓▓▓  (네모 몸통)     ▓▓▓▓▓▓▓▓▓▓
   ▓▓   (짧은 다리2)     ▓▓  ▓▓  ▓▓  (다리4)
```

### v3 기술 사양
- 그리드: **20x20** (v2: 16x16)
- 표시 크기: **160px** (v2: 128px)
- 픽셀 크기: 8px 유지
- 배경: #1a1a19 (약간 밝게)
- 글로우: 리더 + 어두운 캐릭터에 추가

## v3 평가 결과 (2026-04-02)

### 전체: 50점/100 (v2: 75점에서 후퇴!)

| 문제 | 상세 |
|------|------|
| 얼굴 위치 엉망 | 눈/입 중심 안 맞음, 템플릿 좌표 오류 |
| 귀여움 없음 | 퍼피 늘어진 귀 → 뿔/악마처럼 보임 |
| 해상도 낮음 | 7px 픽셀 크기가 뭉개져 보임 |
| 퍼피즈 실패 | 소/악마 같지 강아지 아님 |

### 버전별 순위
1. **v1 (55점)** — 가장 귀엽고 깔끔! 메인으로 사용 추천
2. **v2 (75점)** — 크기/아이템 좋지만 실루엣 비슷
3. **v3 (50점)** — 실패작, 접근 방식 한계

### CSS Canvas 픽셀아트의 한계
- 수동 좌표 입력 → 얼굴 위치 오류 빈번
- 16~20px 그리드로는 정교한 표정 불가
- 캐릭터마다 다른 실루엣 = 각각 새로 그려야 함 → 비효율적

### 다음 단계 권장: 프로급 접근
1. **Icons8/Flaticon**: 햄스터/강아지 아이콘 활용 → 색상 커스텀
2. **Canva 디자인**: 벨라님 Canva에서 직접 배치 → 최고 품질
3. **AI 이미지 생성**: Claude Artifacts 또는 DALL-E로 각 캐릭터 생성
4. **SVG 방식**: 벡터로 그리면 해상도 무한 + 정교한 곡선 가능

> **결론**: v1을 메인 바탕화면으로 사용하면서, 나중에 Canva/Icons8로 프로급 버전 제작!
